# Helinox Wear Morning Dashboard: AI 기반 브랜드 인텔리전스 대시보드

> **브랜드 정의**: "헬리녹스 웨어(Helinox Wear)"는 캠핑기어 브랜드 헬리녹스(Helinox)의 라이선스를 코오롱인더스트리 FnC부문(코오롱FnC)이 취득하여 전개하는 **아웃도어 어패럴** 브랜드이다. 공식 명칭은 "헬리녹스 웨어"이나, 사내에서 "헬리녹스"로 혼용하여 부르기도 한다.
>
> **브랜드 컨셉**: *"기술과 미학이 교차하는 경량 설계"* — 이 컨셉이 고객에게 얼마나 소구되는지를 대시보드 전반에서 추적한다.

## 문제
> HQ 전략팀장과 브랜드매니저(BM)가 리뷰 클리핑·경쟁사·트렌드 데이터를 정확하고 빠르게 확인할 수 없어, 인사이트 도출과 액션 실행이 지연된다

- **현재 상태**: 헬리녹스 웨어의 E-biz팀이 무신사·29CM·자사몰·코오롱몰 4개 채널의 리뷰를 수동 클리핑하느라 시간이 과도하게 소요. 경쟁사 데이터·트렌드 정보가 정확하게 취합되지 않음. 수집된 데이터에서 Insight → Action Item까지 연결되는 체계가 없음.
- **원하는 상태**: 매일 아침 한 페이지에서 AI 브리프 + 채널별 리뷰 감성 분석 + 경쟁사 동향 + 검색 트렌드 + 플랫폼 랭킹 + 크림 인기 지표를 확인. 데이터에서 자동으로 Insight와 Action Item까지 도출.
- **성공 기준**: 🚩 *Red Flag — 구체적 수치 목표는 추후 재논의 필요*
  - (1) 리뷰 클리핑 시간 대폭 절감 → 대시보드 접속으로 대체
  - (2) 경쟁사·트렌드 데이터의 정확성 확보
  - (3) 데이터 → Insight → Action Item 자동 연결

## 헬리녹스 맞춤 설정 상세

### 1. 오늘의 브리프 (Daily Brief) — Option C 확정
- **위치**: 대시보드 최상단, 다크 네이비(#1a1a2e) 배경
- **내용**: 전 섹션 데이터를 크로스 분석한 AI 요약 **4줄**
  - 🛡️ **리스크**: 지금 바로 확인해야 할 이슈
    - 예: *"무신사에서 고어텍스 재킷 봉제 마감 부정 리뷰 2건 발생, 퀄리티 부정률 10% 상승 주의"*
  - 📈 **기회**: 놓치면 안 되는 기회 포착
    - 예: *"'경량 바람막이'(+32%) 검색 급증, 봄 시즌 경량 아우터 마케팅 선점 권장"*
  - 🎯 **액션**: 오늘 당장 움직여야 할 것 1가지
    - 예: *"아크테릭스 무신사 SS 기획전 25% 할인 대응 — 헬리녹스 웨어 봄 신상 프로모션 검토"*
  - 📊 **크림 프리미엄**: 브랜드 가치 지표
    - 예: *"택티컬 스몰 숄더백 발매가 대비 +18% 프리미엄 유지, 거래량 전주 대비 +25%"*
- **갱신 주기**: 매일 07:45 KST 자동

### 2. 채널별 리뷰 요약 (Review Summary)
- **모니터링 채널** (정량 4개):
  - 무신사
  - 29CM
  - 자사몰 (helinoxwear.com)
  - 코오롱몰 (kolonmall.com)
- **표시 항목**: 채널명 | 신규 리뷰 수(배지) | 총 제품 수 | 날짜 탭(최근 5일)
- **데이터 소스**: Google Sheets에 Review Clipper가 매일 07:45 KST 리뷰 적재

### 3. 감성 분석 (Sentiment Analysis)
- **분석 차원** (5차원):
  - 🎨 **디자인**: 외관·색상·핏·실루엣 관련
  - ⚙️ **기능성**: 방수·방풍·보온·통기성·경량성 관련 *(기존 '사용성'에서 아웃도어 어패럴 특화로 변경)*
  - ✨ **퀄리티**: 소재·봉제·내구성·마감 관련
  - 💰 **가격**: 가성비·가격 만족도 관련
  - 📦 **서비스**: 배송·CS·포장 관련
- **브랜드 컨셉 소구 추적**: 리뷰에서 *"기술과 미학이 교차하는 경량 설계"* 컨셉 관련 키워드(경량, 가벼운, 디자인 좋은, 기능적, 테크니컬 등)가 언급되는 비율을 별도 지표로 추적
- **표시 방식**:
  - 제품별 카드: 위험(🔴 빨강 < 80%) / 주의(🟡 노랑 80~89%) / 안전(🟢 초록 ≥ 90%)
  - 주간 전체 요약: 5차원 각각의 긍정 비율(%) + 부정 키워드 리스트
  - "전 28개 제품 모두 보기" 확장 버튼
- **AI 엔진**: Claude Haiku 증분 분석 (신규 리뷰만 분석 → Google Sheets `AI_Sentiment` 탭에 캐시)

### 4. 주간 종합 액션 아이템 (Weekly Action Items)
- **표시 위치**: 브리프 바로 아래 (최우선 확인 영역)
- **구조**:
  - 우선순위 배지: `HIGH`(빨강) / `MID`(노랑) / `LOW`(회색)
  - 카테고리 태그: `감성 분석` / `검색 트렌드` / `리뷰 분석` / `경쟁사` / `플랫폼` / `크림`
  - 담당 팀 태그: `기획팀` / `생산팀` / `영업팀` / `마케팅팀` / `E-biz팀` / `디자인팀`
  - 기한 표시: `D-3` 등
- **생성 로직**: 전 섹션 크로스 분석 → Claude가 팀별 4~6건 우선순위 행동 지시 자동 생성
- **헬리녹스 웨어 예시 액션**:
  - `HIGH` `감성 분석` `생산팀` — "고어텍스 재킷 봉제 마감 부정 리뷰 2건, 긴급 품질 점검"
  - `HIGH` `검색 트렌드` `마케팅팀` — "'경량 바람막이'(+32%) 급증, 봄 시즌 경량 아우터 캠페인 실행"
  - `MID` `경쟁사` `기획팀` — "아크테릭스 SS 기획전 25% 할인 대응 전략 수립"
  - `MID` `크림` `E-biz팀` — "택티컬 숄더백 프리미엄 +18% 유지, 리스톡 물량 검토"
  - `LOW` `리뷰 분석` `디자인팀` — "트레킹 팬츠 핏 관련 긍정 리뷰 급증, 다음 시즌 라인 확대 검토"
  - `LOW` `플랫폼` `영업팀` — "무신사 아웃도어 카테고리 순위 하락, 입점 프로모션 협의"

### 타이포그래피
- **폰트**: Pretendard 또는 Inter (산세리프, Tech 감성)
- **제목**: Bold + 대문자 혼용, 강인한 인상
- **수치 데이터**: Bold + 큰 사이즈 + 싸이언블루 강조
- **본문**: Regular, 가독성 우선

### UI 컴포넌트
- **카드 시스템**: 둥근 모서리(`border-radius: 12px`) + 은은한 그림자의 화이트 카드
- **상태 배지**: 위험(🔴) / 주의(🟡) / 안전(🟢) 색상 + 캡슐형
- **우선순위 배지**: HIGH(빨강) / MID(노랑) / LOW(회색)
- **브랜드 하이라이트**: 헬리녹스 웨어 순위·지표는 **싸이언블루로 강조**
- **확장/축소(아코디언)**: 제품별 리뷰 상세, 전체 제품 목록
- **날짜 탭**: 최근 5일 날짜 탭 UI (리뷰 요약 섹션)
- **즐겨찾기**: 우상단 ⭐ 별 아이콘
- **호버 효과**: 싸이언블루가 은은하게 비치는 절제된 애니메이션

## 데이터 파이프라인

### 리뷰 데이터 수집 (크롤링 자동화 포함)
1. **리뷰 크롤러** (본 프로젝트에서 함께 개발): 매일 07:45 KST 자동 실행
2. **무신사·29CM·자사몰(helinoxwear.com)·코오롱몰(kolonmall.com)** 4개 채널에서 신규 리뷰 크롤링
3. Google Sheets `Reviews` 시트에 적재
   - 컬럼: 날짜, 채널, 제품명, 제품 카테고리, 리뷰 내용, 별점, 작성자

### 크림(KREAM) 데이터 수집
1. **크림 크롤러** (웹 크롤링, 공식 API 없음): 매일 자동 실행
2. 헬리녹스 웨어 제품의 체결가·프리미엄(%)·관심수·찜수·거래량 수집
3. Google Sheets `KREAM` 시트에 적재

### AI 감성 분석
1. 대시보드 로드 시 Google Sheets에서 신규 리뷰 읽기
2. 이전 분석 캐시(`AI_Sentiment` 탭) 확인 → 증분 분석만 수행
3. Claude Haiku가 5차원(**디자인·기능성·퀄리티·가격·서비스**) 분석
4. **브랜드 컨셉 소구** 키워드(경량, 가벼운, 기능적, 테크니컬 등) 언급 비율 별도 산출
5. 결과를 `AI_Sentiment` 탭에 캐시로 저장
6. 프론트엔드에서 캐시된 결과를 렌더링

### 뉴스·트렌드 수집
1. **Naver 뉴스**: 경쟁사 10개 브랜드명 + 아웃도어 어패럴 키워드로 검색
2. **Naver DataLab**: 6개 카테고리(아우터·상의·하의·잡화·브랜드·시즌) 키워드별 주간 트렌드 조회
3. **29CM/무신사 랭킹**: 크롤링으로 아웃도어 카테고리별 인기 순위 수집
